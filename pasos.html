Manual de cómo se hizo mi sitio web

Objetivo

Sitio web

Realizar un sitio web que muestre información de manera sencilla para su fácil manejo y aprendizaje para cada uno de los estudiantes de ingenieria en sistemas y obtener el conocimiento base de cómo crear una página web.

Herramientas utilizadas

Son:

Visual Studio Code, HTML, JavaScript, CSS, GitHub Pages / Netlify e investigaciones para el manejo del tema.

Intrucciones a seguir

Crear un sitio web que muestre lo siguiente:

1. Hoja de vida del alumno 2. Tabla con Nombres y notas de primer y segundo parcial de los cursos cursados actualmente. 3. Galeria de imagenes en el formato que se desee, debe tener al menos diez imagenes, pueden ser imagenes al azar o del alumno. 4. Una pagina que muestre algun video de Youtube (puede usar el ejemplo dado en clase). 5. Una pagina que muestre algun juego utilizando Javascript (debe investigar como colocar un juego sencillo, nada complicado). 6. Una pagina que muestre los pasos que se siguieron para hacer el sitio (tipo manual tecnico).

Paso 1: Planificación

Elección de herramienta de trabajo.

Primeramente se busco en donde era más conveniente trabajar mi sitio web, en la cual se decidió utilizar Visual Studio Code por comodidad.

Paso 2: Estructura HTML y JavaScript

Se creó la base con etiquetas HTML y un poco de JavaScript.

Se usaron etiquetas como <header>, <strong>, <p>, <div>, <section>, y <button>, y se utilizó JavaScript para agregar un juego al sitio web.

Paso 3: Estilos CSS

Se agregaron diversidad de estilos para dar una mejor apariencia visual al sitio web.

Se aplicaron colores, fondos animados y diseños para el aspecto atractivo del sitio con estilos en el <style>.

Paso 4: Imagenes

Se agregaron 10 imagenes.

Se muestran 10 imagenes parte de las maravillas del mundo moderno, en las que se agregaron etiquetas <img>, .

Paso 5: Notas de cursos

Se agregaron las notas de los cursos.

Se muestran las notas obtenidas en los primeros y segundos parciales en cada uno de los cursos de la ingenieria en sistemas, usando la siguiente etiqueta sucesivamente <td class="p1">15>, .

Paso 6: Hoja de vida

Se agrego la hoja de vida.

Se muestra la hoja de vida la cual se realizo utilizando la etiqueta <strong>, .

Paso 7: Video

Se agregaron videos al sitio web.

Se muestran dos videos en el sitio para el entretenimiento de las personas, se utilizo la etiqueta <src, inframe>, .

Paso 8: Juego

Se agregaron un juego usando JavaScript.

Se muestra un juevo el cual es el ahorcado, el mismo fue hecho usando script y etiquetas como <div, mensajediv, const>, .

Paso 9: Marcos

Se agregan marcos

Se agregan marcos para el mejor aspecto visual de mi sitio web.

Paso 10: Manual de uso

Manual de la realizacion del sitio web.

Se muestra un manual el cual fue realizado para dar a conocer la manera en que se llevo acabo el presente sitio web en la que se usaron etiquetas como <p, h2, class, buttos, section, div>, .

Paso 11: Publicion del sitio web

Se publica el sitio web urilizando GitHub Pages para luego compartir la URL.

Conclusión

Crear un sitio web como introducción al mundo tecnológico es algo de suma importancia ya que gracias a ello se aprende muchas cosas que en el transcurso de la carrera serán de gran utilidad.
